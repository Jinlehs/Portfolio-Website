Projects

A summary of my CS/ Design projects


Doctors on Hand Mobile Application

Swift, Java, PHP, MySQL

With the help of Dr. Ching Tan's development team, we created an
iOS and Android telemedical mobile app for the purpose of streamlining
medical consultations and processes. Some of the features include:
Twilio based video consultations, on-demand Ambulance service using GPS
technology, an E-store for prescription drugs and a patient recommendation
system. We created a fully functioning client website, webadmin portal,
and database that connects with the application as well.


Untangler iOS App

Swift

A game that was created with Swift. The purpose of this game is to untangle
the puzzle where all of the lines not intersecting one another. As the game
progresses, levels get harder and harder as more connection nodes are added.


Session 99 Event Website

HTML, CSS, PHP, MySQL

a website that was created for the purpose of informing participants and
spectators of the Session 99 local hip-hop event that takes place every
year. The website is connected to a google calendar that is shows the
events that are taking place this month and a subscription panel so that
people can sign up for Session 99's newsletter. The website also includes
a contact tab where users can send questions to the administrators.


DNA Sequencing Program

C

Using a two dimensional iterative algorithm I created a C program that
will take in two DNA strands 10000 basepairs long and find the longest
common, tandem and palindromic sequences of the two strands.
